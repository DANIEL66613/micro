Como utilizar o endpoint de Alunos

Listar Todos os Alunos

Faça uma requisição GET para a rota:

/alunos

Criar Novo Aluno

Faça uma requisição POST para a rota:

/alunos

Corpo da requisição (JSON):

{
    "nome": "Teste Aluno",
    "idade": 21,
    "turma": "3B",
    "nascimento": "2003-04-15",
    "nota_primeiro_semestre": 7.5,
    "nota_segundo_semestre": 8.0,
    "nota_final": 7.75
}

Editar Aluno

Faça uma requisição PUT para a rota:

/alunos/<id_aluno>

Corpo da requisição (JSON):

{
    "nome": "Teste Aluno",
    "idade": 21,
    "turma": "3B",
    "nascimento": "2003-04-15",
    "nota_primeiro_semestre": 7.5,
    "nota_segundo_semestre": 8.0,
    "nota_final": 7.75
}

Excluir Aluno

Faça uma requisição DELETE para a rota:

/alunos/<id_aluno>
